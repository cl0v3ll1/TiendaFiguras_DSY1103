# TiendaFiguras_DSY1103
Proyecto de FullStacks

Descripción del proyecto

Sistema de tiendas de figuras con funcionalidades de despacho, CRUD de figuras, clientes, tienda, preventas, empresa,
pedido, desarrollado utilizando Spring boot, MySQL y validado en Postman para su correcto funcionamiento

Integrantes:
    Javier Canales Espinoza
    Constanza Barrera Gallegos

Pasos para ejecutar:
    1.- Ejecutar CreateTable para la creación de base de datos
    2.- Iniciar los proyectos para la creación de tablas mediante código
    3.- Ejecutar Los inserts para la inserción de columnas a las tablas creadas
